# 📧 MailSentry Insight

MailSentry Insight is a Flask-based email analytics application that connects to Gmail, fetches emails, classifies them into categories, detects phishing attempts, extracts useful information, and displays analytics through a web dashboard.

## 🚀 Features

- Gmail Integration
- Email Fetching
- Email Classification
- Phishing Detection
- Analytics Dashboard
- CSV Export
- User-Friendly Web Interface

## 🛠️ Technologies Used

- Python
- Flask
- HTML
- CSS
- JavaScript
- Pandas
- Matplotlib
- IMAP (Gmail)
